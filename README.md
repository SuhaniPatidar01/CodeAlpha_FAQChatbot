# 🤖 AI FAQ Chatbot

An AI and Technology FAQ Chatbot built using Python, Streamlit, and NLP techniques.

## 📌 Features
- Answers AI and technology related questions
- Uses NLP and cosine similarity
- Simple and interactive UI using Streamlit
- Handles unknown questions intelligently

## 🛠 Technologies Used
- Python
- Streamlit
- Scikit-learn
- NumPy

## 🚀 How to Run

1. Install dependencies:
pip install streamlit scikit-learn numpy

2. Run the app:
streamlit run app.py
